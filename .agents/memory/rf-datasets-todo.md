# RF Drone Datasets за изтегляне

## 1. RF UAV — Zenodo (2023)
- URL: https://zenodo.org/records/10223214
- Съдържание: DJI M100 дрони, USRP X310, I/Q семпли, anechoic chamber
- Теглене: директно, без акаунт
- Статус: не е теглен

## 2. Drone RF Dataset — KU Leuven (2024)
- URL: https://rdr.kuleuven.be/dataset.xhtml?persistentId=doi%3A10.48804%2FHZRVNZ
- Съдържание: 2024 г., белгийски университет
- Теглене: директно
- Статус: не е теглен

## 3. Drone Remote Controller RF Signal — IEEE DataPort (Open Access)
- URL: https://ieee-dataport.org/open-access/drone-remote-controller-rf-signal-dataset
- Съдържание: RF сигнали от контролера на дрона (не самия дрон)
- Теглене: Open Access = с безплатен IEEE DataPort акаунт (вече е направен)
- Статус: не е теглен

## 4. VTI_DroneSET_FFT — Mendeley Data
- URL: https://data.mendeley.com/datasets/s6tgnnp5n2/3
- Съдържание: FFT данни от сръбски военен институт (VTI), множество дрони
- Теглене: директно от Mendeley (като DroneRF)
- Статус: не е теглен

## Бележки
- CardRF (IEEE DataPort) е Standard tier — платен, пропускаме
- IEEE DataPort акаунтът вече съществува от предишна сесия
- Dataset-ите се пазят на Linux сървъра (192.168.100.224, user: tmm), не на Pi
