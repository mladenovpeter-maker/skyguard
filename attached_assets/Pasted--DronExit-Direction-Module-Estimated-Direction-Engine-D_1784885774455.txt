# DronExit – Direction Module (Estimated Direction Engine)

## Цел

Direction Module е интелигентен слой, който обединява информацията от всички налични сензори и показва на потребителя **най-вероятната посока**, от която идва дронът.

Модулът **не твърди, че знае точните GPS координати на дрона**, а изчислява и визуализира вероятен азимут и степен на увереност (Confidence).

Основната цел е потребителят да получи бърз и разбираем отговор:

> "От коя страна на имота вероятно се намира дронът?"

# Архитектура

```
HackRF Scanner
        │
Wi-Fi Monitor
        │
BLE Remote ID
        │
Signal History
        │
Device Fingerprint
        │
────────────────────────
 Direction Module
────────────────────────
        │
 REST API
        │
 React Dashboard
```

Direction Module работи като независим backend service.

Той не управлява хардуер.

Получава само информация от останалите модули.

---

# Входни данни

Direction Module получава информация от:

### RF Scanner

- frequency
- bandwidth
- RSSI
- signal duration
- signal stability
- detected protocol
- ML fingerprint score

---

### Wi-Fi Scanner

- Vendor
- MAC Prefix
- RSSI
- Channel
- Frame Count
- Detection Confidence

---

### BLE Remote ID

Ако е наличен:

- Drone ID
- Position
- Speed
- Heading
- Altitude

При наличие на валиден Remote ID той има най-висок приоритет.

---

### История

Последните N секунди наблюдения.

Например:

```
13:21:01
RSSI -62

13:21:02
RSSI -58

13:21:03
RSSI -55

13:21:04
RSSI -56
```

Това позволява да се филтрират случайни импулси.

---

# Основна задача

Direction Module НЕ изчислява GPS координати.

Той изчислява:

```
Most Likely Direction
```

пример:

```
North

North-East

East

South-East

South

South-West

West

North-West
```

или

```
315°

40°

170°
```

ако в бъдеще бъде добавена по-прецизна оценка.

---

# Confidence Engine

Всеки резултат има Confidence Score.

Пример:

```
Direction

North-East

Confidence

92%
```

или

```
Direction

West

Confidence

54%
```

При ниска увереност интерфейсът показва:

```
Direction uncertain
```

вместо подвеждаща стрелка.

# Карта

На картата НЕ се показва фиктивна GPS точка.

Показва се сектор.

Например:

```
          N

     ███████

W               E

          S
```

Колкото по-висока е увереността,
толкова по-тесен е секторът.

# Бъдещи версии

Direction Module трябва да бъде разработен модулно.

Версия 1.0

- Обединяване на RF, Wi-Fi и BLE данни.
- Confidence Score.
- Estimated Direction.
- REST API.
- Dashboard Widget.

Версия 2.0

- Поддръжка на допълнителни антени или нови RF входове.

Версия 3.0

- Поддръжка на множество DronExit станции.
- Обединяване на наблюденията между сензори.
- По-прецизни оценки при подходяща инфраструктура.

---

# Основен принцип

Direction Module никога не трябва да показва измислена точка на картата.

Неговата задача е да представя честно и разбираемо най-вероятната посока, базирана на наличните наблюдения, като винаги визуализира и нивото на увереност в оценката.

Това прави DronExit надежден продукт за реални потребители и позволява модулът да се развива във времето без промяна на потребителския интерфейс.
